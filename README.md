# 🏥 Hospital Management System (Odoo 17)

An advanced hospital and clinic management module built with **Odoo 17** and **OWL Framework**. This module provides a modern UI for managing patients, appointments, and prescriptions with an integrated interactive dashboard.

## 🚀 Key Features

- **Patient Management**: Comprehensive database for patient records, including specialized views for pediatric (Kids) patients.
- **Appointment System**: Streamlined scheduling and status tracking for medical visits.
- **Prescription Management**: Digital prescriptions linked directly to patient appointments.
- **OWL Dashboard**: A high-performance, real-time dashboard built using the **Odoo Web Library (OWL)** for data visualization.
- **Security & Access Control**: Pre-configured user groups and access rights (Security Rules).

## 🛠 Tech Stack

- **Backend**: Python 3.12+, PostgreSQL (Odoo Framework).
- **Frontend**: JavaScript (OWL), XML, SCSS.
- **Environment**: Ubuntu 24.04 (Noble), Odoo 17.0 Community/Enterprise.
